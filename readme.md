Для тестирования работы алгоритмов:
1. В main.cpp реализована запись входных параметров (матрицы A) и выходных параметров (матриц Q и R) в файлы `.\test_data\matrix<name>_<size>.txt`.
2. В scilab-скрипте производится проверка обработанных моим алгоритмом данных на совпадение с полученными реализацией scilab QR разложения. Скрипт:
    * ищет данные для матриц с размерами: 100, 200, 300, 400, 500, 1000, 1500, 2000, 2500, 3000 и 3500
    * для всех матриц A запускает собственную реализацию разложения, вычисляет Q и R
    * вычисляет ошибки для A и Q*R
    * вычисляет ошибки для вычисленных Q,R и полученных из файлов Q,R

Для запуска Scilab скрипта:
1. Открыть Scilab
2. Выполнить exec('<путь до репозитория>\QR-decomposition\scilab_script.sc', -1)
